



# arena

## Description


Print the contents of an arena.

Default to the current thread's arena.
## Usage:


```bash
usage: arena [-h] [addr]

```
## Positional Arguments

|Positional Argument|Help|
| :--- | :--- |
|`addr`|Address of the arena.|

## Optional Arguments

|Short|Long|Default|Help|
| :--- | :--- | :--- | :--- |
|`-h`|`--help`||show this help message and exit|
